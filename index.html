Brasil

Ferramentas utilitárias para desenvolvedores Brasileiros

 O que é?

Brasil é um framework JavaScript para node.js e para o browser, que visa facilitar a vida do desenvolvedor de software brasileiro fornecendo a implementação de diversas rotinas comuns, tais como: validação e formatação de CPF/CNPJ, EAN (código de barras), contas e agências dos principais bancos, chaves de acesso de NFe, CEP dentre outros.

O módulo brasil também oferece um conjunto de dados necessários para quem pretende trabalhar com NFe, por exemplo, relação dos códigos CFOPs, relação dos municipios incluindo código IBGE, relação de feriados, dentre outros.

Para os usuários da biblioteca em ambiente node.js também está disponível uma funcionalidade de consulta de CNPJs na receita federal. Veja a demonstração on-line clicando aqui!

 Instalação

Node.js

npm install --save brasil

Browser

bower install --save brasil

Ou você pode baixar diretamente do GitHub na página de releases.

 Utilização

Node.js

var brasil = require('brasil');

Browser

var brasil = window.brasil;
